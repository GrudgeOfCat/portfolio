Solids

For the solids project, we created 6 Platonic and Archimedean solids using AutoCAD. The shapes created in AutoCAD were used to print out and assemble into 3D forms. Surprisingly, this is the most difficult project in this class mainly due to the fact that I was having difficult cutting and assembling the shapes precisely. This projects prepared us for the AutoCAD skill for the Tessellation project later in class.

  1. Tetrahedron
  2. Octahedron
  3. Dodecahedron
  4. Cube
  5. Icosahedron
  6. Truncated Cube
